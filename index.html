Welcome to ENG Language

ENG is a beginner-friendly, English-like programming language designed for simplicity and readability. You write code the way you think, using natural language constructs like let x be 5 or if x is 10.

Features:

  • Readable syntax using plain English
  • Variables, conditionals, loops, and functions
  • REPL and script-based execution
  • JSON saving/loading of variable states
