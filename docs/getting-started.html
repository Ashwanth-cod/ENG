Getting Started with ENG

Welcome to ENG — a programming language that reads like English. This guide will help you install ENG, write your first program, and run it locally or online.

1. Install Python

ENG is interpreted using Python. Install it from the official site:

  • Download Python for Windows, macOS, or Linux
  • During installation, enable the option "Add Python to PATH"
  • After installing, confirm with:
python --version

2. Download the ENG Interpreter

Download the official ENG interpreter script:

Download interpreter.py

3. Write Your First Program

Create a new file called hello.eng and add this:

say "Hello, world!"

This tells ENG to print a message. It's that simple.

4. Run Your Program

Open a terminal in the same folder and run:

python interpreter.py hello.eng

Or on Linux/macOS:

python3 interpreter.py hello.eng

5. Try Online

No installation? Try ENG in your browser:

Launch Web Playground
