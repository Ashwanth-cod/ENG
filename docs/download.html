Download the ENG Interpreter

ENG is designed for simplicity and readability. You can run it locally by downloading the Python-based interpreter.

1. Download the Interpreter

Click below to download the latest version of the ENG interpreter:

Download interpreter.py

2. Install Python

The interpreter requires Python 3.7+ to run.

  • Visit python.org/downloads
  • Install Python and check Add to PATH during setup
  • Verify it works by typing python --version in your terminal

3. Running ENG Code

Once downloaded, you can write an ENG program in a file like hello.eng, and run it:

python interpreter.py hello.eng

If you're on macOS/Linux, try:

python3 interpreter.py hello.eng

4. No Python? Use the Playground

You can try ENG directly in your browser:

Open Web Playground
