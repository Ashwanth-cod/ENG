ENG Syntax: Conditionals

Conditionals in ENG allow your program to make decisions. Use if, else, and logical expressions to control the flow of execution.

Basic If Statement

let x be 5

if x is greater than 3:
  say "x is big"
end

This prints "x is big" because the condition is true.

If-Else

let x be 2

if x is greater than 5:
  say "Greater"
else:
  say "Smaller or equal"
end

Outputs "Smaller or equal" in this case.

Else If (Chained Conditions)

let score be 75

if score is greater than 90:
  say "Excellent"
else if score is greater than 60:
  say "Good"
else:
  say "Try again"
end

Multiple conditions can be evaluated in sequence using else if.

Supported Comparison Operators

  • is – equality check
  • is not – inequality
  • is greater than – greater than
  • is less than – less than
  • is greater than or equal to
  • is less than or equal to

Logical Operators

  • and – both conditions must be true
  • or – either condition can be true
  • not – negates the condition
if x is greater than 3 and x is less than 10:
  say "x is in range"
end
