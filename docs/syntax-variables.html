ENG Syntax: Variables

Variables in ENG are declared using simple, natural language: set <name> to <value>.

Basic Variable Declaration

set name to "Alice"
set age to 30

This creates two variables: name (a string) and age (a number).

Using Variables

say name
say age

Math with Variables

set x to 5
set y to 10
set total to x + y
say total

Updating Variables

set count to 1
set count to count + 1
say count

Variable Types

  • Strings: "hello"
  • Numbers: 42, 3.14
  • Booleans: true, false
