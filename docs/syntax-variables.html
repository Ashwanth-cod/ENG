Variables

Variables let you store values for use later in your program.

Basic Syntax

Use let <name> be <value> to create a variable:

let age be 25
let name be "Alice"
let pi be 3.14

Usage

After declaring a variable, you can use it in expressions:

say name
say age + 5

Reassigning

To update a variable, just use let again:

let age be 30

Dynamic Typing

ENG variables are dynamically typed, so you can assign any type of value (numbers, strings, etc.).
