ENG Syntax: Functions

Functions in ENG allow you to group reusable logic and run it with different values. They're simple and written in plain English.

Defining a Function

define greet(name):
  say "Hello " + name
end

This defines a function called greet that takes one parameter: name.

Calling a Function

greet("Alice")
greet("Bob")

This will print:

Hello Alice
Hello Bob

Returning a Value

define add(a, b):
  return a + b
end

set result to add(5, 3)
say result

This prints: 8

Function with No Parameters

define say_hello():
  say "Hi there!"
end

say_hello()

Tips

  • Function names use letters, numbers, and underscores
  • You can return any type: number, text, boolean
  • Functions must end with end
